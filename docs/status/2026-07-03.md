## Current feasibility verdict

The branch now has a concrete, measured interim answer to the extension program's
north-star question:

| target | current verdict | reason |
|---|---|---|
| Games | Not the right primitive as a core renderer yet | The dynamic-everything requirement is not met; only constrained one-bounce primary-visibility cache splicing is proven. |
| Animated film | Viable component | Static-set animated-light preview and residual identity are promising, but neural animated-camera support and production final-frame trust remain unproven. |
| Feature VFX | Viable component | Per-shot relighting and art-direction loops fit the workflow, but richer learned light embeddings and arbitrary proxy-conditioned controls are still missing. |
