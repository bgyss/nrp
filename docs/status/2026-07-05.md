  - learned polynomial attenuation proxy.
- `tests/test_production_controls.py` adds coverage for:
  - `BasisControlProxy` held-out linear-feature prediction;
  - bad-input rejection for basis-control fitting;
  - soft-mask clipping and mask application helpers.
- `docs/performance.md` reports the E8 toy run over a 32x32 / 12 spp cache with
  24,576 segments and records:
  - exact light-linking algebra for the toy layer partition;
  - binary linking proxy with exact active/inactive predictions and 0 max abs versus
    gather;

  - 4-basis soft link mask held-out PSNR of 331.25 dB with 5.55e-17 max abs;
  - quadratic distance attenuation held-out PSNR of 323.22 dB with 1.94e-16 max abs;
  - 158x speedup for the quadratic attenuation proxy versus polynomial gather in
    the recorded toy run.

  toy-scale conditioned proxies rather than only a narrow gather-time fallback.
